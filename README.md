# Site "Wonders"

## What's Inside
1. Adaptive layout
2. Burger menu for mobile navigation
3. Increment with limit 12

## Installation
1. Clone repository: `git clone https://github.com/dariasnegireva/wonders_daria_snegireva.git`
2. Open `index.html` in any browser
